# Stable Diffusion 應用專案總覽

負責企業內部的 Stable Diffusion 教學與導入，應用於 Logo 風格化、風格轉換、草稿上色與補筆等場景。整合 SDXL、ControlNet、Inpaint 與 LoRA 微調，建立多種可控式生成流程並提供團隊使用。

## 主要亮點
- **系統化教材**：從基礎原理到實務工作流，涵蓋文生圖、圖生圖、LoRA、ControlNet 與提示詞策略。
- **可重用工作流模板**：提供多組 ComfyUI / WebUI 的繪圖模板及提示詞集合。
- **多案例示範**：從 Logo 風格化到跨風格轉換皆提供流程示例，便於複製與調整。
- **品質優化策略**：示範採樣器設定、CFG / steps 調整、LoRA 權重配置與 ControlNet 引導方式。

## 內容概覽
### 工作流與工具使用
- **ComfyUI 流程示意**：以主流程為核心，視需求加入圖生圖、LoRA 或 ControlNet 模組。
- **WebUI 操作指引**：包含模型切換、Negative Prompt 模板、常用 sampling 區間與效果解讀。
- **模型管理**：整理 Checkpoint / LoRA / ControlNet，建立統一命名與版本管理方式。

### LoRA 微調與風格控制
- 微調範圍設定與資料準備方法。
- LoRA 權重調整技巧，避免過度風格化或角色漂移。
- 多 LoRA 疊加時的常用配置策略。

### 生成案例與生產流程
- **Logo / 風格變體**：示範如何利用工作流快速產生不同風格版本。
- **草稿上色 / 補筆**：搭配 ControlNet 與 Inpaint 提升可控性。
- **批量產出**：利用預設模板與提示詞集合，加速大量圖片生成流程。

## 使用與部署建議
- **快速出圖**：先載入既有工作流與提示詞，再依需求微調模型與 LoRA。
- **提高可控性**：複雜構圖或草稿可搭配 ControlNet、Inpaint 並適度降低 denoise。
- **維持風格一致**：同主題使用固定 LoRA 與 Negative Prompt 模板，必要時微調 LoRA 權重。
- **團隊導入**：集中管理模型版本，並在工作流中明確標註可調參數，降低協作成本。
