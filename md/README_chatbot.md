# Chatbot 專案總覽

開發具角色扮演能力的 LLM Agent，涵蓋人格設定、世界觀、語氣風格、多語回覆與內容安全機制。可應用於遊戲聊天室，維持長對話一致性與角色沉浸感，提升互動品質。

## 主要亮點
- **角色化 LLM**：自訂人格與世界觀，維持一致語氣並支援多語回覆。
- **記憶系統**：整合長對話記憶、背景檢索與情緒狀態，使角色互動更自然。
- **內容安全**：敏感詞過濾與不當內容處理，支援前置與後置策略。
- **群聊支援**：自動分派訊息至適合角色，提升群聊協作與沉浸感。

## 功能模組概述
### 玩家特質生成與模擬
- 根據大量聊天紀錄萃取使用者特徵，建立可用於模擬的玩家設定。
- 結構化產生個性描述、語氣習慣與互動偏好，可直接作為模型提示。
- 提供查詢單人或多人互動的基礎 API。

### 角色聊天代理（多版本迭代）
- 區分日常與知識型對話，並可整合檢索流程。
- 維護角色情緒與人格，使對話呈現自然行為變化。
- 具備內容驗證與安全守門，確保回覆品質與一致性。
- 支援對話記憶寫回，用於後續生成連貫性。

### 群聊 NPC 系統
- 支援多角色同時運作，具 @提及、主題偵測、隨機互動等策略。
- 自動維持各角色語氣一致性，並處理多語情境。

### 多角色池管理
- 管理角色的獨立狀態（情緒、人格、歷史與背景）。
- 採用檢索與語氣控制，提供豐富而一致的角色反應。
- 內建語言偵測與角色合法性檢查。

## 部署與應用建議
- 依場景選擇玩家建模、角色模擬或群聊 NPC 模式。
- 記憶與檢索模組可串接多種向量庫或外部模型。
- 建議搭配內容安全模組，確保品牌風格與互動品質。
- 透過批次角色生成可快速擴增角色池與 NPC 多樣性。
