# OCR 文字識別專案總覽

以公開 OCR 模型為基礎，針對特殊字體與不規則排版進行調整，並建立可支援多語系的內部辨識服務。整合傾斜校正、去噪、二值化等前處理，提升複雜影像的辨識率與穩定度。

## 主要亮點
- 支援中文與混合版面，透過再訓練提升對特殊字體與變形排版的辨識能力。
- 前處理包含傾斜校正、去噪、二值化與影像裁切，提高清晰度與召回率。
- 端到端流程涵蓋資料匯入、前處理、模型推理、再訓練與多語部署。
- 支援批次處理，可彈性切換模型版本與字庫。

## 內容概覽
### 端到端流程
- **影像分割與裁切**：將票據、表單等影像拆分成文字區域，提高辨識準確度。
- **前處理管線**：灰階化、二值化、噪聲處理與形態學操作，依場景調整強度。
- **偵測與辨識推理**：使用 OCR 偵測模型找出文字框，再由辨識模型輸出文字結果，支援多語與批量處理。
- **模型再訓練**：依字庫與資料量調整模型輸出空間，並可進行輕量微調以適應特殊場景。
- **批次工具**：提供自動化腳本串接前處理、推理與結果輸出，加速大規模任務。

### 資料來源與擴充
- 支援實拍影像與裁切素材，並可搭配合成資料產生器擴增特殊字體或長尾字形。
- 可依場景建立測試集，持續評估模型效果。

## 使用與部署建議
- 面對特殊字體或複雜版面時，先以小規模資料微調，再逐步擴大訓練集。
- 依影像品質調整前處理強度：輕度噪聲以保留細節為主，嚴重傾斜則應加強校正。
- 部署時注意模型版本、字庫與硬體環境一致性；多語場景可切換不同字庫或模型配置。
- 批次處理流程可搭配日誌與檔案監控，便於長期運行與錯誤追蹤。
